# All-in-One Surface Wiring

Version: 1.0  
Status: Applied  
Scope: Cross-route UI wiring for core, admin, and review navigation

## What this pass does
- Wires all visible route tabs to real working pages.
- Adds a shared shell and reusable UI surface system.
- Upgrades Accounts into a usable list/detail flow.
- Replaces raw placeholders with coherent data-backed surfaces for journal, alerts, payouts, calendar, settings, and backups.

## What this pass does not claim
- It does not replace the canonical missing domain screens where they do not yet exist.
- It does not invent real persistence for unfinished features.
- It does not alter Step 12 reporting or deterministic simulation doctrine.
